- Hot Module Replacement (HMR)

- Source maps enabled

### Production Mode
- Compiled renderer in `out/renderer/`
- Compiled main process in `out/main/`
- Compiled preload in `out/preload/`
- Minified and optimized
- No source maps
- Error logging only
