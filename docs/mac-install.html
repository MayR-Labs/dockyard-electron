üì¶ Manual Installation

Method 1: Helper Script

  1. Download Dockyard-darwin-{arch}.zip from GitHub Releases
  2. Extract the ZIP file
  3. Open Terminal and run: 
    curl -sSL https://raw.githubusercontent.com/MayR-Labs/dockyard-electron/main/scripts/mac-first-run.sh | bash

Method 2: Manual Commands

  1. Download and extract the ZIP from Releases
  2. Open Terminal in the folder containing Dockyard.app
  3. Remove quarantine flags: 
    xattr -cr Dockyard.app
  4. Move to Applications: 
    mv Dockyard.app /Applications/
  5. Launch it: 
    open /Applications/Dockyard.app

Method 3: Right-Click Open

  1. Download and extract the ZIP
  2. Move Dockyard.app to Applications
  3. Right-click on Dockyard.app
  4. Select "Open" from the menu
  5. Click "Open" in the security dialog
  6. Subsequent launches work normally

üÜò If macOS Still Blocks It

  1. Try to open Dockyard (it will be blocked)
  2. Go to System Settings ‚Üí Privacy & Security
  3. Scroll down to see "Dockyard was blocked"
  4. Click "Open Anyway"
  5. Confirm in the next dialog
