üöÄ Installation Steps

Step 1: Download the DMG

Download Dockyard-{version}-{arch}.dmg from GitHub Releases

  • Apple Silicon (M1/M2/M3): darwin-arm64.dmg
  • Intel Macs: darwin-x64.dmg

Step 2: Install the App

  1. Open the downloaded DMG file
  2. Drag Dockyard.app to the Applications folder
  3. Close the DMG window

Step 3: Remove Quarantine (Required)

Option A: Automatic (Recommended)

curl -sSL https://raw.githubusercontent.com/MayR-Labs/dockyard-electron/main/scripts/mac-first-run.sh | bash

Option B: Manual

xattr -cr /Applications/Dockyard.app

Step 4: First Launch

  1. Go to Applications folder
  2. Right-click on Dockyard.app
  3. Select "Open" from the menu
  4. Click "Open" in the security dialog
  5. Subsequent launches work normally (just double-click)

üÜò If macOS Still Blocks It

  1. Try to open Dockyard (it will be blocked)
  2. Go to System Settings ‚Üí Privacy & Security
  3. Scroll down to see "Dockyard was blocked"
  4. Click "Open Anyway"
  5. Confirm in the next dialog

üîê For Advanced Users

If you want to tell macOS to trust Dockyard specifically:

sudo spctl --add /Applications/Dockyard.app
